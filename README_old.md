# AIha Studio Landing

Одностраничный Flask-лендинг для студии прикладной AI-интеграции в производственные, хозяйственные и инфраструктурные процессы.

## Запуск

```bash
cd aiha_landing
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Откройте: http://127.0.0.1:5000

## Что уже есть

- Тёмный glassmorphism-стиль под референс.
- Логотип на овальном белом фоне: `static/img/aiha-logo-oval.png`.
- Обновлённая смысловая структура лендинга без публичного блока «Инструменты подключения».
- Блок «Как проходит внедрение» с пунктом про интеграции в рабочую среду.
- Блок «Как начинается работа» перед формой заявки.
- Форма заявки и виджет обратного звонка для голосового общения.
- Сохранение заявок в SQLite: `leads.db`.
- Место для подключения CRM, Telegram-бота, сервисов уведомлений и закрытого AI-контура.

## Где подключать интеграции

В маршруте `POST /api/leads`, после `lead_id = save_lead(data)`.
